# A/B Тест: Изменение Расположения Рекламного Блока на Маркетплейсе

**Финальный Отчет об Анализе**

---

## 1. Описание Эксперимента

### Цель
Оценить влияние изменения расположения рекламного блока в приложении маркетплейса на ключевые метрики пользователей и выручку компании.

### Исходные Условия
- **Компания:** Маркетплейс товаров
- **Гипотеза:** Изменение расположения рекламного блока увеличит уровень конверсии и среднюю выручку на пользователя (ARPU)
- **Период тестирования:** Данные из реального A/B эксперимента
- **Статистический уровень значимости (α):** 0.05
- **Статистическая мощность (1 - β):** 0.80

### Ожидаемый Эффект
- **Базовая CR (Control):** ~7.91%
- **Целевая CR (Test):** ~8.14% (относительный прирост +2.87%)
- **Минимальный обнаруживаемый эффект (MDE):** 2.87%

## 2. Дизайн Эксперимента

### Группы Тестирования

| Параметр | Контрольная Группа | Тестовая Группа |
|----------|-------------------|-----------------|
| **Описание** | Старое расположение рекламы | Новое расположение рекламы |
| **Переменная в данных** | ab_group = 'control' | ab_group = 'test' |
| **Размер выборки** | 222,334 уникальных пользователей | 222,334 уникальных пользователей |
| **Достаточность** | ✅ Достаточно | ✅ Достаточно |

### Основные Метрики

#### Метрика 1: Коэффициент Конверсии в Добавление (CR_add)
- **Формула:** (Клиенты с adds) / (Клиенты с views) × 100%
- **Интерпретация:** % пользователей, которые добавили товар из рекламы после просмотра главной
- **Тип метрики:** Первичная (главная)
- **Ожидаемый эффект:** Увеличение на 2.87%

#### Метрика 2: Коэффициент Конверсии в Клик (CTR)
- **Формула:** (Клиенты с clicks) / (Клиенты с views) × 100%
- **Интерпретация:** % пользователей, которые кликнули на рекламу после её просмотра
- **Тип метрики:** Вторичная

#### Метрика 3: Коэффициент Конверсии в Заказ (CR_order)
- **Формула:** (Клиенты с orders) / (Клиенты с clicks) × 100%
- **Интерпретация:** % пользователей, которые совершили заказ после клика на рекламу
- **Тип метрики:** Вторичная

#### Метрика 4: ARPU (Average Revenue Per User)
- **Формула:** Общая выручка / Общее количество пользователей с просмотром
- **Интерпретация:** Средняя выручка на одного пользователя, который видел рекламу
- **Тип метрики:** Бизнес-метрика
- **Единица:** Денежные единицы (руб.)

#### Метрика 5: ARPPU (Average Revenue Per Paying User)
- **Формула:** Общая выручка / Количество пользователей с заказами
- **Интерпретация:** Средняя выручка на одного покупателя
- **Тип метрика:** Бизнес-метрика
- **Единица:** Денежные единицы (руб.)

## 6. Методологические Замечания

### Статистические Тесты
- **Для CR (доля):** Критерий независимости хи-квадрат (χ²)
- **Для ARPU/ARPPU (среднее):** t-тест Уэлча (не предполагает равенство дисперсий)
- **Уровень значимости:** α = 0.05 (стандартный для бизнес-анализа)
- **Направление теста:** Двусторонний (проверяем наличие любых различий)

### Допущения
1. Пользователи случайно распределены между Control и Test группами
2. Данные независимы (один пользователь = одна строка)
3. Нет взаимного влияния между группами (не нарушается в offline-тесте)
4. Период тестирования достаточен для стабилизации поведения

### Ограничения
- ARPU и ARPPU близки к порогу значимости — рекомендуется дополнительное тестирование
- Возможное наличие daily seasonality не учитывалось в основном анализе
- Когортный анализ требует дополнительного исследования

## 7. Приложение: Используемые Формулы

### Коэффициент Конверсии
\[ CR = \frac{\text{Количество конверсий}}{\text{Общее количество попыток}} \times 100\% \]

### Стандартная Ошибка для Доли
\[ SE = \sqrt{\frac{p \times (1-p)}{n}} \]

где p — доля, n — размер выборки

### t-статистика для сравнения средних
\[ t = \frac{\bar{x}_1 - \bar{x}_2}{\sqrt{\frac{s_1^2}{n_1} + \frac{s_2^2}{n_2}}} \]

### Минимальный обнаруживаемый эффект (MDE)
\[ MDE = (z_{\alpha/2} + z_{\beta}) \times \sqrt{\frac{p(1-p)}{n}} \times 2 \]

где:
- z_{α/2} = 1.96 (для α = 0.05, двусторонний тест)
- z_β = 0.84 (для мощности 80%)
- p = базовая доля
- n = размер выборки на группу
